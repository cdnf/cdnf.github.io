指定版本升级

如果不想安装升级插件，也可以通过安装指定版本进行插件升级。

首先可以通过 npm outdate 命令查看有哪些版本，然后通过 npm install package_name@package_version 指定安装对应程序及版本，如果不指定版本，则相当于将指定程序下载最新版本重新安装一遍来实现升级。
